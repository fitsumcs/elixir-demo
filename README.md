# Elixir Practice Repo

This repository contains small Elixir exercises and scripts.

## Current Structure

- `01/basic.exs` - simple starter script that prints a message.

## Prerequisites

- [Elixir](https://elixir-lang.org/install.html) installed locally.

Verify installation:

```bash
elixir --version
```

## Run the Script

From the repository root:

```bash
elixir 01/basic.exs
```

Expected output:

```text
Hello world!
```

## Notes

- This is a good minimal starting point.
- Next steps could include variables, functions/modules, pattern matching, and simple tests with `ExUnit`.
